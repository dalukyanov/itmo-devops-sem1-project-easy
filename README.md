# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе

Данный проект запускается на проверку на GitHub

## Установка и запуск

Локально приложение запускается запуском скриптов:
./scripts/prepare.sh
./scripts/run.sh

## Тестирование

Директория `sample_data` - это пример директории, которая является разархивированной версией файла `sample_data.zip`

Приложение тестируется запуском скрипта:
./scripts/tests.sh
